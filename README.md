# Btoor Task

Modern task management interface or productivity dashboard built with clean, responsive design. Features elegant UI components and smooth interactions perfect for task tracking applications. [attached_file:1]

## Features

- Clean, professional task management layout
- Responsive design for all devices
- Modern card-based interface
- Smooth hover effects and transitions
- Mobile-optimized navigation
- Perfect for productivity apps and dashboards

## Tech Stack

- HTML5 Semantic Structure
- CSS3 Grid & Flexbox Layouts
- Custom CSS Animations
- Vanilla JavaScript Interactivity
- Mobile-first Responsive Design

## Quick Start

1. Clone the repository

```
    git clone https://github.com/HossamMohammed2000/btoor-task.git
```

2. Open `index.html` in your browser

3. Start managing your tasks!
